# Manual de usuario (rápido)

## 1) Dashboard

- Presenta accesos rápidos a los módulos.

## 2) Registro de envío

- Completa los datos del envío, remitente y destinatario.
- Presiona **Guardar envío**.
- El sistema genera:
  - Código `ENV-YYYY-0001`
  - Estado inicial `Registrado`
  - Historial inicial
  - QR en PNG (se muestra en un modal)

## 3) Seguimiento / Trazabilidad

- Ingresa el código (ej. `ENV-2026-0001`) y presiona **Buscar**.
- Revisa:
  - Datos del envío
  - Estado actual
  - Historial de estados
- Para actualizar:
  - Selecciona estado
  - Ingresa observación (opcional)
  - Presiona **Guardar cambio**

## 4) Geolocalización + QR

- Busca por código.
- Registra dirección y coordenadas.
  - También puedes hacer click en el mapa para completar lat/long.
- Presiona **Guardar ubicación**.
- Presiona **Generar QR** si necesitas regenerarlo.
